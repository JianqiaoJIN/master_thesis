Granger Causality Analysis for Time Series Data
----------------------



##### Run Experiments:

First generate the simulation system:

    python generator.py

Run bivariate-GC, conditional-GC, group Lasso-GC:

    Rscript run.py
    
Run MLP Lasso-GC:

    python run.py
    
Evaluate the result:

    python evaluate.py
